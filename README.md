# Usage
<!-- usage -->
```sh-session
$ npm install -g unity-package-cli
$ up COMMAND
running command...
$ up (-v|--version|version)
unity-package-cli/0.0.1 darwin-x64 node-v12.18.2
$ up --help [COMMAND]
USAGE
  $ up COMMAND
...
```

```
  $ up help [COMMAND]

ARGUMENTS
  COMMAND  command to show help for

OPTIONS
  --all  see all commands in CLI
```
